# AltCAT

📋 0. Prerequisites
- Node.js 설치 (v23.3.0 권장).
- npm 또는 yarn 설치 (npm 권장).

 🛠️ 1. Clone Project
```bash
git@github.com:snuhcc/AltCAT.git
```

📦 2. Install Dependencies
```bash
cd front
npm install
```

🚀 3. Run
```bash
npm run dev
```

# Backend  (FastAPI)
📋 0. Prerequisites
- Envrionment Setup 수행

🚀 1. Run
```bash
cd backend/app
uvicorn main:app --reload
```
🧪 2. Test
- 127.0.0.1:8000 접속
